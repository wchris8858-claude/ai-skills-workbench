# 敏感词速查表

## 绝对化用语

| 敏感词 | 替换建议 |
|-------|---------|
| 最好 | 优质、推荐 |
| 第一 | 领先、前列 |
| 唯一 | 独特、特色 |
| 最佳 | 优选、精选 |
| 绝对 | 非常、相当 |
| 100% | 高达、接近 |
| 永远 | 持久、长期 |

## 争议话题

避免涉及：
- 政治相关
- 宗教话题
- 地域歧视
- 性别对立
- 未经证实的健康声明

## 平台特定规则

### 抖音
- 避免提及竞品名称
- 不使用"抖音"二字引导
- 避免过度商业化表述

### 小红书
- 不使用"买家秀"等暗示交易的词
- 避免明显的广告标识词
- 不使用虚假测评词汇

### 视频号
- 避免政治敏感内容
- 不使用低俗词汇
- 避免诱导老年人消费
